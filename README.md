# School Event Management System

## Project Overview
The School Event Management System is a frontend web application developed using React.js to simplify the management of school events. The system provides separate interfaces for Students and Organizers, allowing users to access event-related information through an easy-to-use dashboard.

The application helps organize event schedules, event participation, volunteer registration, and feedback collection in a structured manner.

---

## Features

### Student Module

* Student Login
* View Available Events
* Participate in Events
* Volunteer for Events
* View Event Schedules
* Submit Feedback
* View About Page

### Organizer Module

* Organizer Login
* Manage Event Information
* Assign Event Schedules
* View Feedback Section
* View About Page

## Modules

### Home Page
The Home Page serves as the main entry point of the School Event Management System. It provides users with an overview of the application and allows easy navigation to different sections. The page contains introductory information, highlights of the system, and links to Student and Organizer login pages. It is designed with a simple and user-friendly interface to ensure smooth accessibility for all users.

### Events
The Events Management module is responsible for displaying event-related information to users. It presents event details such as event name, event description, event date, and participation options. Students can view available events and register or volunteer for activities. This module helps users stay informed about upcoming events and encourages active participation in school activities.


### Schedule
The Schedule module manages and displays event schedules in an organized format. It provides details regarding event dates, timings, and planned activities. Organizers can assign schedules, while students can easily access timetable information. The module ensures that event information is presented clearly, helping users avoid scheduling conflicts and stay updated with event timings.


### Feedback

The Feedback module allows students to provide opinions, ratings, and suggestions regarding events. Users can submit feedback.This feature supports continuous improvement of future events by encouraging user engagement and communication.

### About

The About Page provides detailed information about the School Event Management System, including its objectives, features, technologies used, and project purpose. It also contains information about the development team and the academic context of the project. This module helps users understand the goals and functionality of the application.
